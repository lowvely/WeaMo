About WeaMo

WeaMo is a weather monitoring system designed to provide real-time weather data such as temperature, humidity, wind speed, pressure, and cloud formation.

This project aims to help users understand current weather conditions using data collected from sensors and visualized through charts and forecasts.

Developed as part of a Computer Engineering project, WeaMo combines hardware sensing, data processing, and web-based visualization.
